FAQ

Is this legal?

We summarize and transform content with short quoted snippets and link back prominently. We respect robots.txt and honor takedown requests.

Will it hallucinate?

We require citations for load‑bearing statements and show freshness timestamps. Readers can vote and flag issues.

Which docs are supported?

Most public technical docs (AWS, Google Cloud, Azure, Stripe, Twilio, etc.). Gated content may require pasting text.

How do versions work?

We store each run with model & prompt tags. Visitors see a random version; votes rank them; you can pin the best.
